# HO6 — Reusable Skill + No-Code Automation

> Hands-on portfolio project · **Week 3** · **Peer-reviewed** · module M10. Part of the **ForgeMind AI — AI Productivity Essentials** course.

## Goal

**Done when:** No-code (Cowork + a Skill) · ≥1 hr/week saved · measured

## What to ship

The Skill + the Cowork automation + a 5-step Loom + a measured before/after time log.

## Pick a problem statement

Choose **one** of these real use-cases — or bring your own (get it approved first):

1. Every Friday you rebuild the same client status report by hand and it eats an hour. Turn your report template into a reusable Skill and have Cowork run it weekly against the latest data, no code. Success: the report generates itself in your template format and you log the before/after time, showing at least an hour a week saved.

2. Your shared inbox or request queue piles up and you triage it manually each morning. Build a Skill that categorises each item and drafts a reply, and have Cowork run it on the queue. Success: incoming items come back sorted with draft responses ready to review, and your time log shows a measurable weekly saving.

3. Your manager wants a metrics deck every month and turning the sheet into slides is tedious. Build a Cowork automation with a Skill that takes the monthly metrics sheet and produces the deck in your format. Success: a finished slide deck generated from the sheet with no manual copy-paste, plus a before/after time log.

4. Action items from your meetings get lost in messy notes and nobody owns follow-up. Build a reusable Skill that turns raw meeting notes into a clean action-item list with owners, run via Cowork. Success: pasting notes returns owned, dated actions every time, and your time log shows the recurring minutes saved.

5. You publish one long piece a week but never have time to repurpose it. Build a Skill plus Cowork automation that turns each long piece into a set of social posts and a newsletter blurb. Success: one input article produces ready-to-post variants in your voice, with a before/after time log proving an hour or more saved.
