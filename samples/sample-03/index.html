metrics deck

Your manager wants a metrics deck every month and turning the sheet into slides is tedious. Build a Cowork automation.

Starter skeleton. The files in this folder are blank templates. Your job is to fill them in using Claude.ai. Read each TODO carefully.

Your task

Build a Claude Skill that turns raw metrics data into a narrative slide-by-slide outline, then connect it to a Cowork automation so the deck is ready before your monthly meeting.

Step 1 — Build the Skill

TODO

  • Open claude.ai → Skills → New Skill
  • Decide what metrics your deck needs to cover (revenue, growth, NPS, etc.)
  • Write the skill instructions in skill-template.md — fill in every [TODO] section
  • Paste the finished instructions into the Claude Skills editor
  • Test it: paste a sample metrics table and run the skill
  • Save the final skill text back into skill-template.md

Step 2 — Build the Cowork automation

TODO

  • Open a Cowork session in Claude.ai
  • Upload or paste your metrics spreadsheet
  • Follow the steps you designed in cowork-playbook.md
  • Copy the slide outline into your slide tool (Google Slides, PowerPoint, etc.)
  • Record your time savings in time-log.csv

Files in this folder

skill-template.md → fill this in first cowork-playbook.md → then this time-log.csv → track your savings

Checklist

  • Metrics list finalised (what KPIs go in the deck)
  • Skill name decided and written at the top of skill-template.md
  • Instructions written (purpose, steps, output format)
  • Example input and example output added to skill-template.md
  • Skill tested manually in Claude.ai with real metrics data
  • Cowork automation steps written in cowork-playbook.md
  • End-to-end automation run at least once
  • Time saving logged in time-log.csv
