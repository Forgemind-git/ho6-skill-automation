client status report

Every Friday you rebuild the same client status report by hand. Use a ready-made Claude Skill to turn rough project notes into a polished, client-ready report — then have Cowork run it for you.

Ready to use. The files in this folder are already filled in — a complete Skill and a complete Cowork playbook. Copy them into Claude.ai, run them, then tweak for your own work. No API key needed.

What this is

A reusable Claude Skill that generates a weekly client status report from bullet-point inputs, plus a Cowork automation that runs it on a schedule. Everything is already written — copy it, try it, then adapt it.

✓ Works with your Claude.ai subscription — no API key

Step 1 — Try it in one chat

  1. Open claude.ai and start a new chat.
  2. Open README.md in this folder and copy The example prompt.
  3. Paste it into Claude and press enter — you'll see the finished result straight away.

Step 2 — Save it as a reusable Skill

  1. In Claude.ai, open Skills → Create Skill.
  2. Name it Weekly Client Status Report.
  3. Copy everything from skill.md into the instructions box and click Save.
  4. In any chat, type /, pick Weekly Client Status Report, and paste your own data.

Step 3 — Make it run itself (Cowork)

  1. Open automation-playbook.md — it's a complete step-by-step recipe.
  2. Follow it to connect your data and schedule the run in Cowork.
  3. Record the time you save in time-log.csv.
